US Equity Markets 2017

About

  • This Data Visualisation Dashboard allows users to analyse and compare NYSE and NASDAQ equities, priced below USD$50, at 2017 year end.

  • The data displayed on each chart is by default presented by sector. By using the drop-down selector each sector can be drilled into, to display the data by industry.

User Guidance

  • Using the Select Industry drop-down selector, choose Select all to display the data on each chart by sector, for all NASDAQ and NYSE equities under USD$50, at 2017 year end.

  • To drill into each sector by industry, use the Select Industry drop-down selector and choose the relevant industry.
