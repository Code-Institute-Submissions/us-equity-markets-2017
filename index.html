US Equity Markets 2017

About

This Data Visualisation Dashboard allows users to analyse NYSE and NASDAQ equities, priced below USD50, as at the end of 2017.

The data displayed on each chart is by default presented by sector. Each sector can be drilled into and the data displayed by industry type.

User Guidance

Using the Select Industry drop-down selector, choose Select all to display data within each chart, for all NASDAQ and NYSE equities under USD50 as at 2017 year end.

To drill into each sector by industry, use the Select Industry drop-down selector and choose the relevant industry.
